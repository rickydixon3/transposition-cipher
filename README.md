# Transposition Cipher GUI (Java Swing)

An application built in Java using Swing that performs transposition encryption and decryption.

The program allows users to enter text, provide a numeric key, and view both the transformed output and the internal matrix representation used in the cipher.


##  Features

- Encrypt text using a transposition cipher  
- Decrypt text using the same key  
- Display the internal 2D matrix used for transformation  
- Clear input/output fields  
- Responsive GUI built with Java Swing
